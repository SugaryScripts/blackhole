###### Install
```sh
npm i -g yarn
```
###### global
```sh
yarn global add
yarn global list
```

###### yarn 4
```sh
# goto project dir
corepack enable
yarn set version stable
yarn set version from sources
yarn install
# if error
# remove yarn prefix on .zshrc
```
###### project
```sh
yarn add -D @vitejs/plugin-vue
yarn remove <package>
yarn outdated
yarn upgrade
yarn upgrade-interactive --latest
```
